# AI-CDSS Project

## 📌 Description
This project is an AI-based Clinical Decision Support System.

## ⚙️ Features
- Predicts disease based on input
- Uses ML model
- Simple web interface (Flask)

## 🛠️ Tech Stack
- Python
- Flask
- Pandas / NumPy

## ▶️ How to Run
1. Clone repo
2. Install requirements
3. Run main.py
